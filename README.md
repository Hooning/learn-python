# Learn Python

A beginner-friendly Python learning repository covering fundamental concepts and syntax.

## Contents

| File             | Topic            | Description                                                      |
| ---------------- | ---------------- | ---------------------------------------------------------------- |
| `0_hello.py`     | Hello World      | Your first Python program                                        |
| `1_print.py`     | Print Statements | Escape characters and formatted output                           |
| `2_variables.py` | Variables        | Variable assignment and f-strings                                |
| `3_input.py`     | User Input       | Reading input from users                                         |
| `4_data_type.py` | Data Types       | Strings, integers, floats, sets, tuples, dictionaries, and lists |

## Getting Started

### Prerequisites

- Python 3.x installed on your system

### Running the Examples

```bash
python 0_hello.py
```

## Topics Covered

- **Print Statements** - Basic output and escape characters (`\n`, `\t`)
- **Variables** - Storing and using data with f-strings
- **User Input** - Interactive programs with `input()`
- **Data Types** - Working with different Python data types:
    - Strings and string methods
    - Integers and arithmetic operations
    - Floats and decimal numbers
    - Complex numbers
    - Sets, tuples, dictionaries, and lists
